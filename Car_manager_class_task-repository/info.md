JS структура:
// 1. КЛАССЫ:
class Vehicle {
    - #fuel (приватное)
    - static totalVehicles
    - constructor(name, fuel)
    - get fuel() / set fuel()
    - drive() — уменьшает топливо на 10
    - refuel() — увеличивает топливо на 20
    - getInfo() — возвращает объект с данными
}

class Truck extends Vehicle {
    - constructor(name, fuel, cargoWeight)
    - drive() — уменьшает топливо на 20
    - refuel() — увеличивает топливо на 30
}

// 2. МАССИВ МАШИН:
const fleet = []; // хранит все машины

// 3. ФУНКЦИИ ДЛЯ DOM:
addVehicle(event)
Читает три значения из формы — название, тип, топливо.
По типу создаёт либо new Vehicle либо new Truck и пушит в fleet.
Потом вызывает renderFleet и очищает поля формы.


renderFleet()
Находит ul в HTML. Очищает его через innerHTML = ''.
Проходит по массиву fleet циклом — для каждого элемента вызывает renderCard
и добавляет результат в ul. В конце вызывает updateStats.


renderCard(vehicle, index)
Принимает объект машины и его индекс в массиве.
Создаёт li элемент. Внутри строит разметку:
название, тип, полоску топлива, цифру топлива, три кнопки.
На каждую кнопку вешает обработчик который передаёт индекс в нужную функцию.
Возвращает готовый li.


removeVehicle(index)
Удаляет из fleet элемент по индексу через splice.
Вызывает renderFleet.


driveVehicle(index)
Берёт объект из fleet по индексу.
Вызывает у него метод drive().
Вызывает renderFleet.


refuelVehicle(index)
Берёт объект из fleet по индексу.
Вызывает у него метод refuel().
Вызывает renderFleet.


updateStats()
Считает длину fleet — это всего машин.
Фильтрует fleet по типу — считает легковых и грузовиков.
Находит три элемента в HTML и вставляет в них числа.


getFuelColor(fuel)
Принимает число.
Возвращает строку с цветом:


зелёный если > 50
жёлтый если от 20 до 50
красный если < 20


Используется внутри renderCard когда строишь полоску топлива.
